# ⚔️ Standoff  
*A fast-paced 2D dueling game built with Godot Engine*  

## 🎮 Overview  
**Standoff** is a local multiplayer 2D combat game developed in **Godot 4** using **GDScript**.  
Players face off across connected arenas, battling to reach the opponent’s base.  
Each duel emphasizes **timing**, **prediction**, and **strategy** rather than combo complexity.

---

## 🧩 Core Features  
- **Local PvP Combat** – Two players fight using melee attacks and parries.  
- **Dynamic Arenas** – Five connected maps with different visuals and layouts.  
- **Combat System** – Light, heavy, and parry attacks linked to animation frames.  
- **Hit / Hurt Logic** – Custom `Area2D` hitboxes and hurtboxes synchronized with animations.  
- **Health & Stamina System** – Signal-based updates for smooth gameplay.  
- **Death & Respawn Flow** – Automatic resets for continuous action.  
- **Optimized Performance** – Stable 60 FPS on standard setups.  

---

## 🛠️ Tech Stack  
- **Engine:** Godot 4.2  
- **Language:** GDScript  
- **Version Control:** GitHub  
- **Assets:** Custom 2D sprites and animations  
- **Testing:** Local multiplayer (keyboard / gamepad)
